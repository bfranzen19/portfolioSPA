software & web development


all of my websites are built completely from scratch. these examples have all been designed, created, and maintained solely by me. all of these sites are built using MongoDB (utilizing Mongoose), express.js, vue.js, node.js, and NPM as well as HTML, CSS, and javascript. all of my sites are fully accessible and responsive to ensure the same experience for every user (visually impaired users, color blindness) on every device (mobile, tablet, desktop, etc.) using bootstrap. the sites below are categorized by live sites (deployed to their own URLs) and other projects with GIFs to show functionality and looks of the applications.

TriggerAware
| |

this database allows you to search by tv/movie title or trigger type to ascertain whether or not there is a risk for you or another survivor to be triggered. users can also fill out a form to recommend triggering scenes to add to the database. this application was built as my final project at CodeCraft and was built using web scraping (cheerio and request), javascript, mongoDB (using mongoose to validate data), express, fuse.js (fuzzy search JS library), bootstrap 4, CSS, HTML, bcrypt for authentication to login and verify the recommended scenes from users.

background

after my time in the army, i started working in GIS and satellite imagery which pushed me to further consider a career change into web and software development. codecraft school of technology made this career change a reality for me.

at codecraft, i studied the javascript MEVN stack which includes mongoDB and mongoose, expressjs, vuejs, and nodejs. i also became proficient in using BASH/command line, git, github, HTML, CSS, jQuery, bootstrap, object oriented programming, and writing both front and backend code.

in addition to my time at codecraft, i have also taken classes in python, java, object oriented programming, software development lifecycle (SDLC), and javascript-based web mapping at the university of denver in denver, colorado.

in the beginning of 2018, i began the #GrowWithGoogle android basics challenge course on udacity. this course allowed me to further practice java, object oriented programming, and learn xml, android studio, and mobile development practices. after the 3 month challege course, i was awarded a full scholarship to the android basics nanodegree through udacity.

after graduating from codecraft, i was offered a position a junior application developer at a thriving, excellent company here in boulder, colorado. i'm happy to be getting paid to do what i love and learning relevant, helpful skills. now, i work mostly in the apex programming language as a salesforce developer. although a lot of my work is comprised of integrating various geographic information systems into our salesforce instance so i do get to write some javascript pretty regularly, and use my GIS background as well.

please take a look at my github account to see what i've been working on and my progress throughout my time as a student at codecraft, as a student in the #GrowWithGoogle android basics course, and as a junior application developer at my current company.
